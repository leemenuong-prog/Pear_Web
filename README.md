# Pear_Web

Pears public roadshow website. It is a static React + Vite site for Netlify.

Live site: https://pear-web-leemenuong.netlify.app

Netlify admin: https://app.netlify.com/projects/pear-web-leemenuong

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run test
npm run build
```

Netlify build command: `npm run build`

Netlify publish directory: `dist`

## Replace Materials

- Agent URL and page copy: `src/content/site.ts`
- Demo video: `public/media/pears-demo.mp4`
- Full deck: `public/deck/pears-roadshow.html`
- Brand mark: code-native `🍐 Pears`; do not use legacy PNG logos as brand identity.
